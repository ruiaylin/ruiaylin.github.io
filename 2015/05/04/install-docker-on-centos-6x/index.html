Install docker on CentOs 6.x

Docker 的安装环境是对内核有要求的，官方也是建议 2.6.32-431 或者更高版本

安装EPEL的仓库

更新lib 包

yum update -y

安装

yum -y install docker-io

启动docker进程

service docker start

Install docker on CentOs 7.x

安装

 yum install docker

关闭防火墙

systemctl stop firewalld
