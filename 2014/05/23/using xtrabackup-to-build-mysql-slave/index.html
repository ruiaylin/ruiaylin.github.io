下载安装XtraBackup

1.在http://www.percona.com/downloads/XtraBackup/LATEST/   
2.解压安装，并配置path路径 export PATH=$JAVA_HOME/bin:/usr/local/xxx:$PATH  
3.我的mysql的配置文件是/xx/my.cnf，如果不指定，XtraBackup默认使用此文
